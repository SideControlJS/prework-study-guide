HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • It is important to use h1 through h6 in order for organization and accessability.
  • The img element should contain an alt attribute that describes what the image is.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding inicates how much space we want around the content inside an element.
  • The Box Model describes that everything in CSS has a box around it.
  • Using a Link element with a rel attribute and your CSS file name is the most common method to incorporate your CSS into your project.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file]: adds a file as it looks now to your next commit
  • git config --global user.name [firstname lastname]: sets a name that is identifiable for credit when reviewing version history
